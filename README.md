# allegro.ts examples

## a HTML5 game making library

---

## [GitHub](https://github.com/alegemaate/allegro-ts) - [API](https://alegemaate.com/allegro-ts/)

### Building

Install packages
`yarn`

Build examples

```sh
yarn build
```
